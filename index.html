CSS Selector Basics

Simple selectors

Syntax Name Example Description
* Universal selector * will match all the elements of the document
elementname Type selector h2 will match any h2 element
.classname Class selector .firstname will match the element that has class="firstname"
#idname ID selector #nickname will match the element that has id="nickname"
A,B Selector list div, span will match all div and all span elements

Combinator selectors

Syntax Name Example Description
A B Descendant selector div span will match all span elements that are inside a div element
A > B Child selector ul > li will match all li elements that are nested directly inside a ul element
A ~ B General sibling selector p ~ span will match all span elements that follow a p, immediately or not
A + B adjacent sibling selector h2 + p will match the first p element that immediately follows an h2 element
